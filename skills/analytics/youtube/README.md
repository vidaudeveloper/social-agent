# YouTube 数据分析

基于 [youtube-analytics-cli](https://github.com/Bin-Huang/youtube-analytics-cli)。

```powershell
npm run youtube:stats-setup
npm run youtube:stats -- videos VIDEO_ID
npm run youtube:stats -- channels UC...
```
